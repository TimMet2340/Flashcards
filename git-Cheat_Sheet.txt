Die Basis-Befehle

git clone [URL]
    -Kopiert ein existierendes Repository (Projekt)

git add [Dateiname]
    - Schickt Änderungen in die "Vormerkliste"
    - git add . fügst du alle geänderten Dateien gleichzeitig hinzu

git commit -m "Nachricht"
    - Speichert deine vorgemerkten Änderungen lokal als festen snapshot

git push
    - lädt deine lokalen Commits auf den Server

git pull
    - holt die neuesten Änderungen vom Server und mergt sie direkt mit deinem lokalen Code


Ablauf
    pull (Updates holen)

    Code schreiben/ändern

    add (Änderungen vormerken)

    commit (Änderungen lokal speichern)

    push (Änderungen hochladen)
